Endpoints

API Endpoints

Direct access points to the Coderic Content Network. All endpoints are publicly accessible.

GET /api/helloWorld LIVE

Test endpoint - Hello World response from Cloudflare Workers

https://coderic.net/api/helloWorld
GET /api LIVE

API index - List all available API endpoints

https://coderic.net/api
GET /v1/content

Retrieve all available content from the network

https://api.coderic.net/v1/content
GET /v1/content/{id}

Get specific content by ID

https://api.coderic.net/v1/content/abc123
GET /v1/datasets

List all available datasets

https://api.coderic.net/v1/datasets
POST /graphql

GraphQL endpoint for flexible queries

https://api.coderic.net/graphql
WS /stream

WebSocket endpoint for real-time streaming

wss://ws.coderic.net/stream

Response Format

All API endpoints return JSON responses:

{
  "status": "success",
  "data": { ... },
  "meta": {
    "timestamp": "2025-11-02T15:00:00Z",
    "version": "1.0"
  }
}

Rate Limiting

Public endpoints have a rate limit of 1000 requests per hour per IP address.
