Endpoints

Endpoints de API

Puntos de acceso directo a la Red de Contenido Coderic. Todos los endpoints son accesibles públicamente.

GET /api/helloWorld ACTIVO

Endpoint de prueba - Respuesta Hello World desde Cloudflare Workers

https://coderic.net/api/helloWorld
GET /api ACTIVO

Índice de API - Lista todos los endpoints de API disponibles

https://coderic.net/api
GET /v1/content

Recupera todo el contenido disponible de la red

https://api.coderic.net/v1/content
GET /v1/content/{id}

Obtiene contenido específico por ID

https://api.coderic.net/v1/content/abc123
GET /v1/datasets

Lista todos los datasets disponibles

https://api.coderic.net/v1/datasets
POST /graphql

Endpoint GraphQL para consultas flexibles

https://api.coderic.net/graphql
WS /stream

Endpoint WebSocket para streaming en tiempo real

wss://ws.coderic.net/stream

Formato de Respuesta

Todos los endpoints de API devuelven respuestas JSON:

{
  "status": "success",
  "data": { ... },
  "meta": {
    "timestamp": "2025-11-02T15:00:00Z",
    "version": "1.0"
  }
}

Límite de Velocidad

Los endpoints públicos tienen un límite de velocidad de 1000 solicitudes por hora por dirección IP.
