Public APIs

Access shared content through our RESTful APIs. All endpoints are publicly available for integration.

REST API

Standard RESTful API endpoints for content access.

GET https://api.coderic.net/v1/content

GraphQL

Query content with GraphQL for flexible data retrieval.

POST https://api.coderic.net/graphql

WebSocket

Real-time data streaming through WebSocket connections.

wss://ws.coderic.net/stream

Public CDN

Content delivery network for fast global access.

https://cdn.coderic.net/assets/

API Authentication

All APIs are publicly accessible. No authentication required for read-only operations.

Example Request:

curl -X GET "https://api.coderic.net/v1/content" \
     -H "Accept: application/json"
